What is blockchain?

Block + Chain = Digital information stored in a public database

Block (contains following)

  • • Data: transaction information, date, dollar amount, etc.
  • • Digital Signature: username
  • • Hashcode: a unique code to distinguish each block
  • • Each can store thousands of transactions

Chain

  • • A chain is blocks strung together
  • • Each block is added linearly and chronologically
  • • Publicly available for EVERYONE
  • • Updates automatically with each new appended block

Network

  • • Node: computers connected to blockchain network
  • • Nodes all see same blockchain
  • • Decentralized: everyone owns the information; No third party is necessary

*** Blockchains can only be updated by consensus between participants in the system, and once new data is entered it can never be erased. It is a write-once, append-many technology, making it a verifiable and auditable record of each and every transaction.

Security

How are blockchains protected?

Decentralized

By design, the decentralized system of blockchain makes it hard to hack. If someone wanted to manipulate information on the blockchain, they’d have to manually change each and every single copy of the blockchain out there.

Hash Codes

Each block contains its own personal hash code and the hash of the previous block. These hashes connect the blocks on the blockchain, so it’s impossible to change one block without changing every other block. That means hacks are very traceable and difficult to carry out.

Consensus Models

To add blocks to a chain, each computer has to pass a test, called consensus models, to verify that the information on the block is trustworthy. One such test is the proof of work system, which is implemented by bitcoin.

Application

What is bitcoin?

Bitcoin is an online cryptocurrency that became popular because it operates on a decentralized authority, meaning it’s untraceable and secure. Blockchain was implemented in 2008 under the pseudonym Satoshi Nakamoto to serve as a public transaction ledger for bitcoin.

How do you mine a bitcoin?

Whenever a new block is added to a blockchain, miners check if the transactions within the block are accurate to prevent "double-spending". In exchange for their work on verifying numerous transactions, miners are awarded bitcoin, or “block rewards”.To do so, they solve a block's highly complex computational math problem, the "proof of work." More specifically, they must come up with a 64-digit hexadecimal number that is less than or equal to a target hashcode that is changed periodically for security. Trying to get that target hashcode is basically guesswork and reliant on how fast your computer can generate that hash.
